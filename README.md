# PRPing

GitHub App that reminds you when an Issue is closed or a PR is merged. Never forget your follow-ups.

## How it works

1. Comment `/remind <memo>` on any Issue or PR
2. PRPing reacts with :eyes: to confirm
3. When the Issue is closed or PR is merged, you get notified with your memo

## Notification channels

- **GitHub comment** — enabled by default
- **Slack** — configure your Incoming Webhook URL on My Page

## My Page

Sign in with GitHub to see your active reminders. Each reminder shows:

- **Status** — Pending (waiting) or Notified (already fired)
- **Trigger** — Issue close or PR merge
- **Memo** — what you asked to be reminded about
